# 认知科学

> *Cognitive Science*

认知科学研究心智与智能的计算本质——感知、记忆、语言、推理与意识如何在生物与人工系统中实现。它是"类脑智能"中"懂心智"的一环，连接神经科学（机制层）与人工智能（实现层）。

## 章节规划

1. **认知科学基础** — 认知科学的历史、范式与方法论。
2. **感知与注意** — 从感觉输入到知觉构建，以及注意的选择与限制。
3. **记忆与学习** — 记忆的多重系统与学习的认知机制。
4. **语言与符号** — 语言作为认知的窗口：从音系到语义到语用。
5. **推理与决策** — 人如何推断、判断与在不确定下选择。
6. **概念与范畴** — 知识如何组织为概念与范畴结构。
7. **意识与自我** — 主观经验、自我意识与心智的"难问题"。
8. **认知的计算模型** — 用可运行的模型刻画认知过程。
